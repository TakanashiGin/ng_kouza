******************************************************************************************************

　　　　　　　　　　　　　　　　　　　ティラノスクリプト用テーマ一括変換プラグイン

******************************************************************************************************

　ダウンロードいただき誠にありがとうございます
　ゲーム制作のお役に立てれば幸いです
　動作確認はティラノスクリプト v474 で行っています

――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
■ファイル詳細
――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
　theme_kopanda_03 は以下の２種類を同梱しております。
　制作されるゲームの画面サイズに合わせてお選びください。

　　theme_kopanda_03a：幅640px、高さ960px（縦長）
　　theme_kopanda_03b：幅960px、高さ640px（横長）


――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
■このプラグインでできること
――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
　ゲームのシステム周りのデザインを丸ごと着せ替えることができます
　変更される箇所は以下の８つです

　・メッセージウィンドウ
　・システムボタン
　・メニュー画面
　・セーブ画面
　・ロード画面
　・バックログ画面
　・コンフィグ画面
　・glinkボタンのデザイン


――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
■プラグインの導入方法
――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
　プロジェクトファイルの data > others > plugin フォルダを開きます
　ここに theme_kopanda_03a あるいは theme_kopanda_03b フォルダを配置します

　シナリオファイルに 

　　[ plugin name="theme_kopanda_03a" ]　縦長版を使う場合はこちら
　　[ plugin name="theme_kopanda_03b" ]　横長版を使う場合はこちら

　と記述するとプラグインを読み込みます
　読み込んだ時点でテーマが反映され画面のデザインが変わります
　ゲーム全体にテーマを反映させたい場合は、first.ks あたりに記述するのがベターです

　【注　意】
　　プラグインを読み込ませた後で [ positon ]タグなどでメッセージウィンドウの設定を変更しないでください
　　レイアウトが崩れるおそれがあります


――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
■システムボタンの表示・消去
――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
　初期の状態ではシステムボタンは表示されていません
　表示させるには [ add_theme_button ] と記述してください
　非表示にしたいときは [ clearfix ]タグを使用します 


――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
■システムボタンのカスタマイズ
――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
　theme_kopanda_03 フォルダの中にある init.ks を編集することでカスタマイズができます
　不要なシステムボタンを消したり、表示する位置を変更したり、自分で作った素材に上書きしたり
　自由に改造していただいてかまいません


――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
■引数について
――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
　プラグインの読み込み時に引数を渡すことができます
　デフォルトで設定しているものは font_color, name_color, frame_opacity の３つです

（例）
　[ plugin name="theme_kopanda_03" font_color="0xFFFFFF" name_color="0xFFCC00"  frame_opacity="255" ]

　font_color：メッセージの文字色を指定できます
　name_color：名前欄の色を指定できます
　frame_opacity：メッセージ枠の透明度を0〜255の範囲で指定して下さい（0で完全に透明、255で完全に不透明）


――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
■glinkの使用について
――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――
　テーマとおそろいの glink ボタンのデザインが使用できるようになります
　color パラメータに ts03a または ts03b と指定すると反映されます
　デザインを改造したいときは ts03a.css または ts03b.ks の最後のほうをいじってやってください

（例）
　[ glink  color="ts03b"  storage="scene1.ks"  size="20"  x="260"  width="400"  y="100"  text="はい。興味あります"  target="*selectinterest"  ]

■おまけ　クリック待ちグリフについて
――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――

　プロジェクトファイルの tyrano > images sysytem フォルダにある nextpage.gif を上書きしてご利用ください

　※念のためバックアップを取っておくことをオススメします
